## Start it
1) Install packages in services and package<br/>
    <p>with npm</p>
    <code>
    $ npm i
    </code>
    <p>with yarn:</p>
    <code>
    $ yarn
    </code>
2) Start server and client and admin
   <p>yarn command:</p>
   <code>$ yarn dev</code>
   <p>npm command:</p>
   <code>$ npm run dev</code>
